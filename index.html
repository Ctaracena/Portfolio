Algo sobre Mí

Hace poco más de un año tomé la decisión de iniciar este viaje de aprendizaje en algo que toda la vida me ha apasionado, el Desarrollo Web.

En este tiempo he tenido la oportunidad de estudiar en la Escuela Platzi, de la mano de los mejores profesores de latinoamerica, así como tambien he tomado cursos en freeCodeCamp, del cuál me he certificado como "Responsive Web Developer" con una práctica de 300 horas. Además de mucha investigación de forma independiente.

En los últimos años me he dedicado a la música, la cual me ha dejado un gran aprendizaje en toda la parte creativa como en el diseño y edición de video, audio, fotografía y demás contenido multimedia.

Mi deseo es continuar aprendiendo cada día y poder poner en práctica el conocimiento adquirido diseñando y creando contenido de calidad que pueda cumplir con las expectativas y garantizar la satisfacción de los clientes.

ESTOS SON ALGUNOS PROYECTOS QUE HE DESARROLLADO

Pasteles Annie

Es un sitio web de tipo e-commerce hecho desde Shopify que incluye un catálogo de productos, una página de preguntas frecuentes, un formulario de contacto para que el cliente proporcione su datos y así poder dar seguimiento de su experiencia como usuario del sitio. Tambíen incluye un carrito de compras y un portal de pago.

Easybank Landing Page

Este proyecto fue desarrollado utilizando únicamente HTML, CSS y Javascript Vanilla para el cambio en el menú de navegación al pasar a tamaño Mobile (dispositivos pequeños). Contiene una Sección principal y dos secciones con varias tarjetas con información importante, la primera con ícons y la segunda con fotografías y por útlimo el píe de página con los links hacia las redes sociales y otros contenidos relacionados.

Este Sitio Web fue desarrollado con WordPress. IR AL SITIO

Simpson's Test

Este proyecto es en su mayoría programación en Javascript, el desafío en este caso fue aplicar la lógica necesaria para hacer que el programa reconozca cuando se escoje la respuesta correcta o incorrecta y según sea el caso ir acumulando un puntaje que al final nos dé un resultado con un mensaje dependiendo del score final. Así como también bloquear las demás opciones al momento de haber escogido una opción, entre otras cosas. Puede ser facilmente adaptable a cualquier tipo de Test.

IR AL SITIO

Secret Message

Este programa crea una codificación de un mensaje convirtiendolo a un grupo de caracteres indescifrables, el cual puede servir para enviar, de manera pública, mensajes de caracter clasificado o "secreto". El programa también sirve para decodificar los mensajes, de manera que si dos personas tiene este programa pueden codificar un mensaje, enviarlo y la otra persona decodificarlo y leerlo.

ALGUNAS DE MIS CERTIFICACIONES

Frontend Developer

Curso de HTML y CSS para definir la arquitectura de código y construir un sitio web usando componentes estáticos. Maquetar las pantallas principales de una página web. Agregar diseño responsivo usando preprocesadores para optimizar el código CSS.

Sistemas de Diseño

Conocer y dominar los sistemas de diseño, una herramienta esencial para equipos de desarrollo y diseño que trabajan juntos en la creación de nuevos elementos de software. Aplicar paso a paso el proceso de implementación a nivel frontend y llevar a feliz término todos los proyectos.

Responsive Web Design

Curso extenso completamente en inglés con alrededor de 300 horas de contenido y práctica, enfocado en aplicar diseño visual, accesibilidad, diseño adaptable a cualquier dispositivo y todo esto con un código limpio y aplicando buenas prácticas.

Fundamentos de Javascript

JavaScript es un lenguaje de programación que se trabaja desde el navegador. Este curso enseña a construir programas, conocer el entorno, los condicionales y las estructuras repetitivas. Aprender cuáles son y cómo se declaran las variables y las funciones de JS.
